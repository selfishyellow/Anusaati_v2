
Anusaati_v2
A discord bot focused upon making users life automated, easier and fun
features-
1. Automated Session Tracking
• Automatic Detection: Automatically parses messages for Forest invite links (English, French, and Korean support included).
• Persistent Sessions: Posts an interactive session card in a designated channel, allowing members to join or mark interest.
• Role Pinging: Can be configured to ping a specific role when a new session starts.
2. User Analytics
• Focus Tracking: Keeps track of each user's total focus minutes.
• Streaks: Calculates and maintains current and longest focus streaks based on daily activity.
• Leaderboard: Provides a server-wide leaderboard to see who has the most focus time.
• Profile Card: View a detailed summary of your own or other users' focus stats, streaks, and favorite plants.
3. Smart Reminders
Personalized Reminders: Allows users to set reminders for upcoming sessions (e.g., "Remind me 5 minutes before").
• Automated Notifications: Sends a DM to the user when their requested reminder time is reached.
4. Interactive Plant Gallery
• Plant Information: Get details and images of specific plants used in sessions.
• Gallery: Browse all available plants in the library via a paginated gallery.
5. Server Configuration
• Customization: Admins can use the /setup command to specify a dedicated channel for sessions and a specific role to ping when a new session is created.
This bot effectively turns simple invite messages into organized community events while providing gamified tracking to keep members motivated.
